# CTF-UTILS
--- 

A NodeJs based ctf-utils package.

## 1. Installation
```cmd
    npm install ctf-utils
```

## 2. Usage

### Detect | Decode | Encode

Detect/Decode/Encode all possible encrypt types of an input context.

|  Types   | Detect | Decode | Encode |
| :-----: | :----: | :----: | :----: |
|  Base64  |   ✅    |   ✅    |   ✅    |
| Unicode  |   ✅    |   ✅    |   ✅    |
|   MD5    |   ✅    |        |   ✅    |
| 当铺密码 |   ✅    |   ✅    |        |
| 与佛论禅 |   ✅    |   ✅    |        |
